Linear Regression finds the best-fit line through data by minimizing the sum of squared residuals (OLS). The model learns weights w and bias b such that ŷ = Xw + b. Gradient descent iteratively updates weights: w ← w - α·∇L where α is the learning rate and ∇L is the loss gradient.

Recurrent Neural Network processes sequences by maintaining a hidden state hₜ = σ(Wₓxₜ + Wₕhₜ₋₁ + b). Each time step takes current input xₜ and previous hidden state hₜ₋₁, producing output yₜ. This enables memory of past inputs — critical for time-series, text, and speech.
Sequence Config

h = 4
Hidden State Gates (LSTM)
F
Forget Gate: fₜ = σ(Wf·[hₜ₋₁,xₜ] + bf) — what to forget from cell state
I
Input Gate: iₜ = σ(Wi·[hₜ₋₁,xₜ] + bi) — what new info to add
O
Output Gate: oₜ = σ(Wo·[hₜ₋₁,xₜ] + bo) — what to output
